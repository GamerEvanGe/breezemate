# PyInstaller VERSIONINFO resource for BreezeMate.exe.
#
# Two reasons we ship this:
#   1. Windows uses the file version + size + hash combo as the cache
#      key for icon thumbnails in Explorer. Bumping the version forces
#      Explorer to re-extract the embedded RT_ICON instead of showing
#      a stale icon from a previous build.
#   2. Right-click -> Properties -> Details now displays meaningful
#      branding instead of empty strings, which looks much more like
#      a real shipped product.
#

# (filevers=, prodvers=, the StringStruct values). The four numbers
# are (Major, Minor, Build, Revision).
